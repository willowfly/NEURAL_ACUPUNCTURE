# NEURAL_ACUPUNCTURE
shared data and code for neural discharge analysis of AA rats during acupuncture

This is the shared raw data and code for the article "Acupuncture-Induced Dorsal Root Potentials and Mast Cell Degranulation in AA Rats" by Wei Yao et al. 

Please start with `technical_report.mlx` or `techinical_report.pdf`.
